# 使用说明
### 1. 设置页面
  1. 设置 [Scrcpy](https://github.com/Genymobile/scrcpy "Scrcpy") 文件夹路径[必须]：
    1. 在输入框`Scrcpy 路径`中输入你的 Scrcpy 路径，***一定要子路径后加上`\` 或 `/`！***
  2. 设置命令行参数[可选]：
    1. 在输入框`命令行参数`中输入参数（最终结果：`scrcpy.exe -s <设备> <命令行参数>`）。

### 2. 开始页面
  1. 无线连接：
    1. 在输入框`无线连接`中输入设备的 IP 地址。
    2. 单击`连接`，***单击后会连接到 ADB***。
